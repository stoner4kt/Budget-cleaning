## 1. GOOGLE ANALYTICS 4 SETUP

### Step 1: Create GA4 Property
1. Go to https://analytics.google.com/
2. Click "Admin" (gear icon in bottom left)
3. Click "Create Property"
4. Enter property name: "Budget Cleaning Solutions"
5. Select timezone: "South Africa"
6. Select currency: "ZAR - South African Rand"
7. Click "Next" and complete setup

### Step 2: Get Your Measurement ID
1. In your GA4 property, go to "Admin" > "Data Streams"
2. Click "Add stream" > "Web"
3. Enter your website URL: https://budgetcleaningsolutions.co.za
4. Name the stream: "Budget Cleaning Website"
5. Copy the **MEASUREMENT ID** (format: G-XXXXXXXXXX)

### Step 3: Update Your Website
**Replace in ALL HTML files:**
- Find: `G-XXXXXXXXXX`
- Replace with: Your actual Measurement ID

**Files to update:**
- index.html
- services.html
- contactform.html
- thank-you.html

### Step 4: Set Up Conversions
In GA4:
1. Go to "Configure" > "Events"
2. Click "Create event"
3. Create these key events:
   - `form_submission` (contact form)
   - `quote_request` (quote button clicks)
   - `phone_click` (phone number clicks)
   - `whatsapp_click` (WhatsApp button clicks)

## 9. PERFORMANCE OPTIMIZATION

### Current Optimizations Implemented:
✅ Semantic HTML5
✅ Accessibility attributes (ARIA labels)
✅ Lazy loading for images
✅ Preconnect for external resources
✅ DNS prefetch for analytics
✅ Defer JavaScript loading
✅ Minified external CSS/JS

### Additional Recommendations:

**1. Image Optimization:**
```bash
# Convert images to WebP format for better compression
# Use tools like ImageOptim, TinyPNG, or Squoosh
```

**2. Enable Compression:**
Add to .htaccess (if using Apache):
```apache
<IfModule mod_deflate.c>
  AddOutputFilterByType DEFLATE text/html text/plain text/xml text/css text/javascript application/javascript application/json
</IfModule>
```

**3. Browser Caching:**
Add to .htaccess:
```apache
<IfModule mod_expires.c>
  ExpiresActive On
  ExpiresByType image/jpg "access plus 1 year"
  ExpiresByType image/jpeg "access plus 1 year"
  ExpiresByType image/png "access plus 1 year"
  ExpiresByType text/css "access plus 1 month"
  ExpiresByType application/javascript "access plus 1 month"
</IfModule>
```

**4. Test Performance:**
- Google PageSpeed Insights: https://pagespeed.web.dev/
- GTmetrix: https://gtmetrix.com/
- WebPageTest: https://www.webpagetest.org/

**Target Scores:**
- PageSpeed: 90+ (Mobile & Desktop)
- First Contentful Paint: < 1.8s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
